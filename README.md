# Aenigma
Aenigma (Latin for puzzle) is a Sudoku puzzle game for Sailfish OS.

## Implemented Features
- Different difficulties (Easy / Medium / Hard / Insane)
- Show errors in grid
- Highlighting
- Undo

## Planned Features
- Feature requests
